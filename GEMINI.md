# Canva Landing Page - Gemini CLI Blueprint

This project is a recreation of the Canva landing page, built with Next.js 14 and Tailwind CSS. It was generated by the Gemini CLI.

## Features

*   **Next.js 14 App Router:** The project uses the latest Next.js features, including the App Router.
*   **Tailwind CSS:** The styling is done with Tailwind CSS, a utility-first CSS framework.
*   **Component-Based Architecture:** The application is divided into several reusable components, making it easy to maintain and scale.
*   **Responsive Design:** The layout is responsive and should work well on all screen sizes.

## How to Run

1.  Install the dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
